# MindBalance

## Опис

MindBalance е веб-апликација инспирирана од HBSC (Health Behaviour in School-aged Children) студијата.

Апликацијата им овозможува на младите да ги следат своите дневни навики и да добијат Wellness Score според нивниот животен стил.

## Функционалности

- Избор на расположение
- Следење на часови сон
- Следење на време поминато на телефон
- Следење на физичка активност
- Следење на ниво на стрес
- Пресметка на Wellness Score
- Персонализирани совети
- Зачувување на резултатите
- Графички приказ на резултатите

## Технологии

- HTML5
- CSS3
- JavaScript
- Chart.js
